Tabs Manager - Auto Group and Save

How to Use Support Privacy Policy

Effective Date

April 24, 2026

Overview

Tabs Manager - Auto Group and Save does not collect, sell, or transmit personal data to external servers.

Permissions and Usage

  • tabs: used to read and organize open tabs for grouping features.
  • tabGroups: used to create, update, and restore browser tab groups.
  • storage: used to store extension settings and saved tab groups.

Data Storage

  • Settings are stored in Chrome sync storage when available.
  • Saved tab groups are stored in Chrome local storage.

All data stays in the browser profile controlled by the user.

Third-Party Services

This extension does not send browsing data to third-party analytics or remote APIs.
