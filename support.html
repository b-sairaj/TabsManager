About the Extension

Tabs Manager - Auto Group and Save helps you organize tabs by domain, save groups, and restore your workflow quickly from the popup or options page.

For Web Store listing, use this page URL as your Support site.

Common Help Topics

Grouping does not run

  • Open chrome://extensions, enable Developer mode, and reload the extension.
  • Check that Enable grouping and Automatically group new tabs are turned on in options.

Saved groups are missing

  • Saved groups are stored in browser local storage.
  • Open the popup and options page once to refresh the current data view.

Domain mismatch after search redirects

  • The extension waits for redirects to settle before grouping.
  • If needed, wait 1-2 seconds and reopen the popup to confirm final grouping.
